# FFS Logistics

## Description

FFS Logistics company website.

## Installation

    ### npm install
    ### npm install react-router-dom@5
    ### npm install react-bootstrap bootstrap
    ### npm install react-icons --save
    ### npm install @emailjs/browser --save
